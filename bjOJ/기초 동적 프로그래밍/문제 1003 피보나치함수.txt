문제 1003 피보나치함수

피보나치 함수를 돌리는데 
0,1이 출력되는 횟수를 구하는 것임

전역변수로 count 선언하고 피보나치함수에서 
0, 1일 때, count++를 해준다

dp= dynamic programming
동적 프로그래밍

하루에 4시간은 해야 손해 XX

하루에 5시간은 해야지 손해 XX

하루에 


방법
dp 즉 동적 프로그래밍을 사용해야 관리가 편하다
->그러기 위해서 2차배열 사용

0일때 0,1이 사용되는 횟수

1일때 0,1이 사용되는 횟수

-또한 피보나치 함수도 배열을 반환하는 함수로
1. 0의 횟수만 고려한 피보나치
2. 1의 횟수를 고려한 피보니치 
를 이용해서 횟수 반환

ex
fibonacci(N-1)[0] ->반환하는 배열이 dp[]라고 할때
오류: dp[0] 값을 반환ㄴㄴ
맞는 것:
fibonacci(N-1)[0] = dp[N-1][0]이다
왜냐면 fibonacci(N-1)은 dp[N-1]을 반환하므로
fibonacci(N-1) = dp[N-1] 이다

여기서 StringBuilder를 사용해서 시간 단축 됨
그냥 출력하면 시간 초과남
(출력을 반복하는 대신 StringBuilder해야 맞음)
